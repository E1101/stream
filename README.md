# Stream
### News Feed & Activity Stream V0.1

#### How to Make
Run 
```sh
$ make
```

and find binaries at ./build/{platform}/{cmd-name}
  
To build for specific platform run _make_ as 
```sh
$ make linux
$ make darwin
$ make windows
```
or make them all together by
```sh
$ make all
```

by default it just make Linux version
